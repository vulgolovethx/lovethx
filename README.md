🧩 Meus projetos:

🔹 lvMinas

Sistema completo de administração de minas.

**Principais recursos:**

* Armazém automático de minérios
* Sistema de Lucky Blocks
* Picaretas customizadas
* Encantamentos personalizados
* Mensagens e menus 100% configuráveis
* Sistema de database MYSQL para servidores grandes

https://github.com/vulgolovethx/LoveMinas

 🔹 lvMinasPV

Sistema avançado de minas privadas (integrado ao lvMinas).

**Principais recursos:**

* Integração com lvMinas
* Sistema de hologramas (dependência: HolographicDisplays)
* Evolução da mina
* Sistema de convite de jogadores
* Abertura da mina com comissão
* Sistema completo de progressão
* Sistema de database MYSQL para servidores grandes

https://github.com/vulgolovethx/lvMinasPV

 🔹 lvCore

Sistema base obrigatório para todos os plugins LV.

**Principais recursos:**

* Sistema de licença personalizado
* Geração de Server-ID
* Autenticação via Key única
* Proteção contra uso não autorizado

https://github.com/vulgolovethx/lvCore
